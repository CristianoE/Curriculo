Gerador de Currículo

Este é um projeto de Gerador de Currículos feito com HTML, CSS (Tailwind), PHP e JavaScript (jQuery).

Funções

- Formulário para preenchimento de dados pessoais
- Adição dinâmica de formações, experiências e habilidades
- Cálculo automático da idade
- Geração de currículo em formato imprimível

Linguagens utilizadas

- HTML5
- Tailwind CSS
- JavaScript (jQuery)
- PHP

Estrutura

- `formulario.html`: Formulário de entrada
- `codigo.php`: Geração do currículo com base nos dados
- `script.js`: Funções JS para adicionar/remover campos e calcular idade
